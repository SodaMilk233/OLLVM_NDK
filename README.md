# 🛡️ OLLVM_NDK for Termux

## 📖 简介
**OLLVM_NDK** 是专为 **Termux** 环境优化定制的 Android NDK，集成了 **OLLVM（Obfuscator-LLVM）** 混淆编译器。此版本让您在 Android 设备上直接进行 C/C++ 代码的混淆编译，提升应用安全性。

## ✨ 特性
- ✅ **轻量便携** - 无需复杂配置
- ✅ **开箱即用** - 解压即可开始使用
- ✅ **集成混淆** - 内置代码混淆功能
- ✅ **双版本支持** - 提供 NDK r28c 和 r29 两个版本

## 📦 可用版本

| 发布名称 | 版本号 | 文件大小 |
|----------|--------|----------|
| android-ndk-r28c | 28.2.13676358 | ~4.7GB |
| android-ndk-r29 | 29.0.14206865 | ~4.3GB |
